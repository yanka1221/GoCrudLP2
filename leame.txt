How to use
You can edit main.go at local or docker container (golang_app)

docker exec -it golang_app bash


 docker run --rm golang go env

Build & Run 

on_golang_app_container
go run main.go

or

on_your_computer
docker exec -it golang_app bash -c "go run main.go"
